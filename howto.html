How to use the learning module?

  1. The learning module will pick questions for you to review based on a modified super memo 2 algorithm.
  2. Read the question, phrase the answer either in your mind or out loud and press the 'show answer' button.
  3. Compare your answer to the one stored in the database. Rate your answer to provide input to the spaced repetition algorithm (the algorithm strives to optimize when to show you the question again, to maximize review efficiency and retention).
  4. It is perfectly fine to use this software for learning. Meaning, it is okay to not know an answer to a question. Also, when you see a question for the first time, the algorithm goes into learning mode - it will be cycling through questions without delay until you answer 'spot on' for the first time. When this happens, only then the spaced repetition algorithm will kick in.

      5. How to select which lectures to see questions from?

      1. To select the lectures from which the learning module will pick questions for you, please go to Settings.
      2. Select lectures.
      3. Hit save.

      How to toggle receving a summary email with questions ready for review?

      1. This email is sent out every three days. To enable / disable it please go to Settings.
      2. Enable or disable the email.
      3. Hit save.
